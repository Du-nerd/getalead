# GetaLead 🎯

AI-powered lead generation for businesses. Find clients every day across Facebook, TikTok, Instagram, WhatsApp and Google Maps.

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) and import the repository
3. Vercel auto-detects Vite — just click Deploy
4. Your live link is ready in ~2 minutes

## Local Development

```bash
npm install
npm run dev
```
